Printer Connection Setup

  • Printer Connection Setup
  • Printer Setup

Keywords: printer setup, wireless printer setup, 123 com setup

Once receiving the printer, assemble the printer to start the printing process. With the advent of technology the printing process has become simple and easy. The user have the option to connect mobile, laptop, computer or even tab with the printer with wired or wireless connection for starting the printing process. Follow the below guidelines to get started with wired and wireless connection method. Connecting printer with network through wired method Initially collect mobile or computer that is connected with the network with wired or wireless method. Then, make sure to turn on your printer and check for the basic requirements with the printer setup. On your computer or mobile, download and install HP Smart application.

  1. Collect the Ethernet cable and check for a stable internet connection.
  2. Connect the Ethernet cable with the printer.
  3. Make sure the mobile or computer connected to the same network as the printer.
  4. Turn on your mobile or computer and open the HP Smart application.
  5. If you don’t have the application, you can download from 123 com setup.
  6. Click the Get Started button and agree to the terms and conditions.
  7. On the following screen, press the plus symbol to add new printer.
  8. From the following list of printer, chose your printer name.
  9. Similarly, for mobile follow the same procedure.
  10. Simply follow the remaining on screen instruction to finish the wired printer setup.

Connecting printer with network through wireless printer setup

Connecting your printer with wireless network allows printing conveniently from your computer, mobile or tab anytime.

  1. Initially restore the printer default settings.
  2. Make sure your printer have enough paper, and ink loaded.
  3. Check if the network and the power is stable.
  4. Move your printer to the router as much as possible.
  5. Remove any radioactive materials in between as it may distract the signals.
  6. Download and install the latest printer driver from 123 com setup.
  7. Once done on your printer, navigate to Wireless and select Setting.
  8. From the following, choose Wireless Setup Wizard option.
  9. From the network list, choose the network name.
  10. Enter the password for the network to complete the wireless printer setup.

Have connected your printer with the network through wired or wireless network but, still facing print issue then follow the below guidelines to quickly troubleshoot the printer setup issues

Wired printer setup troubleshooting:

  1. Initially, check if you have a seamless power supply.
  2. Check the network speed.
  3. Make sure you are using the defect less Ethernet cable.
  4. Try connecting with different Ethernet port and make sure the port is working fine.
  5. On your printer, check if it is having enough ink and paper loaded to start the process.
  6. Especially, check if you are having the latest printer driver downloaded in your printer.
  7. If not, download and install the latest printer setup from 123 com setup.
  8. Then, check if the printer and your device is connected on the same network.
  9. From your printer navigate to the settings and print network configuration page or wireless network test results page to troubleshoot any issue.
  10. If none of the above steps solve your issue, contact the printer support for a quick remedy.

Wireless printer setup troubleshooting:

You can try some basic troubleshooting steps like restarting the printer, looking for enough print material with printer and stable power with network connection. Tried everything but no result? then follow the below guidelines to solve the wireless printer setup connection issue.

  1. Initially print a Wireless Network Test report from the printer control panel to check your printer is connected with the network.
  2. Download and run the HP print and scan doctor application from 123 com setup to check the network connection issue.
  3. Especially, check if you are having the latest printer driver downloaded in your printer.
  4. If not, download and install the latest printer setup from the printer official site.
  5. Check if you have forget to update the latest router settings in your printer,
  6. Similarly, check if you are choosing the proper network name and entering the relevant password.
  7. Don’t access your printer network connection in VPN. Disable the VPN till the end of the process.
  8. From your computer setting, navigate to system preference and choose printers and scanners option.
  9. On the following list, remove your printer and re-add that again with the computer.
  10. Finally check the firewall settings or try disabling them throughout the wireless printer setup.
